ProtocolBuilder

class ProtocolBuilder<T : Any>(classRef: KClass<*>)

The scope wherein a protocol's read and write operations are defined.

If the protocol of a nested class is defined and its package contains any uppercase letters, attempting to read it from binary will throw ClassNotFoundException.

fun fallback(read: TypedReadOperation<T>): TypedReadOperation<T>

When prepended to a read operation, declares that subtypes without a read operation can still be instantiated as an instance of T using this read operation. Generally, this should be used for types whose subtypes have the same public API. Any information not deserialized as a result is lost.

fun static(write: TypedWriteOperation<T>): TypedWriteOperation<T>

When prepended to a write operation, declares that the only information serialized from an instance of T is that which is specifically written here. If used, subtypes of this type may not define a protocol with a write operation. Enables certain optimizations.
